## Recommendation System
This project is a simple Recommendation System using Python.
It suggest relevant Items to users based on their selected preferences such as movies, books, products, and music.
The system demonstrates the basic concepts of content-based recommendation logic usinf predefind datasets. 
## Objective
the main objective of the project is to:
- Understand how recommendation systems work.
- Implement category-based filtering using python.
- Practice dictionary, loops, and user input handling.
- Buid a simple interactive CLI- based application.
## Features
- Movies recommendation
- Books suggestion
- Product recommendation
- Music recommendation
- Continuous user recommendation
-Exit option to terminate program gracefully.
## Working Principle
The system follows a simple logic:
1. A predefind dataset is created using python dictionary.
2. User selects a preference category.
3. The system check wheather the category exists.
4. If found:
   - It displays all items under that category.
5. If not found:
   - It shows an error message.
6. The process continues until the user exits.
## Algorithm
1. Start program
2. Initialize dictionary with categories and items.
3. Display available categories to user.
4. Take user input.
5. if input == "exit" - Stop program
6. If categories exists - display recommendations.
7. Else - show "No recommendations found"
8. Repeat loop
