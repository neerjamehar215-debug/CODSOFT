# rule Based Chatbot
This Project is a simple Chatbot developed using Python. it responds to user messages using predefined rules.
## Objectives 
To Develope a simple rule-Based chatbot using Python and understand basic AI concepts.
## Features
- Responds to Greeting messages.
- Answers Simple Questions.
- Ends Conversation Using bye Command.
## Technology Used 
- Python 
- Visual Studio Code
## How to Run 
1. Open Terminal 
2. Run Python Chatbot.py
3. Start Chatying with the Bot
## Output
User: hello
Bot: hello!
User: how are you
Bot: I am fine.
user: bye
Bot: Goodbye!
