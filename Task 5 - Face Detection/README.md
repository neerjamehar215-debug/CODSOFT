## Face Detection System
The Face Detection System is a Artificial Intelligence and Computer Vision project that detects human faces in images using opencv and Haarcascade Classifier. The system identifies facesand highlights them with rectengular bounding boxes.
## Objectives
- Tdevelop an AI-based Face Dectection System
- To detect an human faces in digital images.
- To apply Computer Vision Techniques in Opencv.
- To understand face detection using Haar scade Classifier.
- To gain practical experience in AI and image processing.
## Features
- Automatic face detection from images.
- Draws bounding boxes around detected faces.
- Fast an efficient face detection. 
- Detect multiple faces in a single image.
- Simple and User-friendly implementations.
## Working Principle 
- The user provides an images an input.
- The image is converted into grayscale format.
- Haar Cascade Classifier scans the image for facial features.
- Bounding boxes are drwan around detected faces.
- Faces are detected based on trained patterns.
## Algorithm
1. Start the program.
2. Load the Haar Cascade face detection model.
3. Read the input image.
4. Convert the image into grayscale.
5. Detect faces using detectMultiscale().
6. Count the detected faces.
7. Draw rectangles around detected faces.
8. Display the output image.
9. End the program.
